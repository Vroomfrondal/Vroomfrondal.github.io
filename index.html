Notely

Notely is a productivity application for taking notes. This multipage app features an editing page, dynamically-updating bodies and titles, and a timestamp feature similar to Instagram's "posted X minutes ago". Start creating your own notes here.

NASA Picture of the Day App

Because of my love for space, as a personal project I built a web app that provides a new astronomy picture or video each day. The app was built using vanilla HTML, CSS, Javascript, and NASA's APOD API. You can also pick a day from the past, all the way back to June 16, 1995 when NASA initialized their photography database. You can test the application yourself here.
