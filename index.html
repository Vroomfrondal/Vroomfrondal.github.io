projects

Tax Calculator

The Cerruti Company Payroll project was a Visual Basic script I developed for one of my classes. The purpose of the application was to take input from the user, in this case name, marital status, and financial information; and the application would return Gross pay, Federal Income tax, Payroll tax, and Net Pay. It also taught me some GUI work and challenged my understanding of accounting principles.

Mock Fitness Website

The Fitness website was my first HTML, CSS, and JS Project. It was a mock representation of something a gym company would use for their online presence. It also helped teach me some of the fundamentals of vanilla front-end languages.

Hangman Game

This was a letter guessing game I developed using Visual Basic. It takes a five-letter word from player one (which is censored), and then player two is to try and guess the word with keyboard inputs.

To Do List App

The To Do List application is an interactive, web browser based productivity tool I developed as a personal project. It uses HTML5, CSS, and Javascript to provide a GUI for scheduling your daily tasks. The app stores data in your web browser's cache so even if you leave the app, you can return later to your list. You can test the application yourself here.
